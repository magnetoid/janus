# Journal 2026-06-09



## 03:09 · decision

Self-learning/proactive upgrade shipped + pushed (642428d, 1f327ff). TWO new features:
(1) PROCEDURE MINING (agent/skill_miner.py, 'janus skills mine'): the procedural twin of memory mining. Reviews a finished session and DRAFTS reusable skills (SKILL.md + optional python script) into skills/.drafts/ for review — never auto-activates. Dedupes vs existing skill names. Closes the gap that today's skill creation is reactive (agent must notice mid-task / hit the nudge).
(2) PROACTIVE ASPIRATIONS (agent/aspirations.py, 'janus aspire set/list/show/done/clear/checkin'): long-horizon north-star goals Janus holds across sessions (distinct from /goal Ralph loop). 'set' drafts a milestone roadmap (aux LLM) + auto-creates ONE idempotent weekly cron (Mon 9am, '0 9 * * 1') running aspire_checkin.py which prints active aspirations into a check-in prompt so the agent proactively asks 'still your goal? next step?'. aspirations.json store.
Both: injectable llm_caller, best-effort, fully tested (9+9 tests). GOTCHA learned: cron parse_schedule does NOT accept 'every monday 9am' — use cron expr '0 9 * * 1'. Composes with session mining for progress detection.

Interest-driven self-learning (capstone) shipped + pushed (02da2cf). agent/interests.py + 'janus interest add/list/remove/discover'.
